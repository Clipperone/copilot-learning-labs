# Agent Persona Definitions

This folder holds role-specialized agent definitions with purpose statements, tool permissions, and handoff protocols.

## When This Folder Gets Populated

You create agent definitions during [Lab 06 — Agents and Personas](../labs/lab-06-agents-and-personas/). The lab produces 10 definitions. The capstone adds at least 3 more scoped to your project context.
