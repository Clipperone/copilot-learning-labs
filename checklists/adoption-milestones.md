# Adoption Milestones Tracker

Use this tracker alongside `capstone/roadmap.md` to record your actual progress at each milestone. Update it monthly.

This is a live document — do not fill it in all at once. Record your state at each checkpoint when you reach it.

---

## How to Use This Tracker

1. At each milestone date, open this file and update the **Actual state** column.
2. Compare against your gate deliverable from `capstone/roadmap.md` Section 2.
3. If the gate is not met, record what is missing and when you will close it.
4. Commit the updated file — this is your adoption audit trail.
