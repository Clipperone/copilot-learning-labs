# Beginner Completion Checklist

Use this checklist after completing Modules 01–03 and Labs 01–03 to confirm you are ready for the intermediate level.

---

## Module 01: Foundations

- [ ] Copilot Pro+ is active — no status bar errors
- [ ] Can switch between Ask, Plan, and Agent modes (Edit mode was consolidated into Agent in VS Code 1.110)
- [ ] Can trigger and dismiss inline completions and inline chat
- [ ] Can explain — in one sentence each — when to use each mode
- [ ] Applies the four critical review questions to AI-generated code

## Lab 01: Getting Started

- [ ] All success criteria from [labs/lab-01-getting-started/checklist.md](../labs/lab-01-getting-started/checklist.md) are met
- [ ] Premium model access confirmed (or limitation documented)

---

## Module 02: Configuration

- [ ] `.vscode/settings.json` committed with `useInstructionFiles: true` and format-on-save
- [ ] `.github/copilot-instructions.md` has at least 3 specific, verifiable rules
- [ ] Copilot reflects the conventions when asked in Ask mode
- [ ] A linter is running on save in VS Code

## Lab 02: Project Configuration Baseline

- [ ] All 6 tasks complete — configuration files created and verified
- [ ] All 4 tests pass via the VS Code test task

---

## Module 03: Token Optimization

- [ ] Can classify any Copilot action as included or premium without looking it up
- [ ] Can select the correct mode and model for any task type in under 5 seconds
- [ ] Can write a compact prompt with goal, constraints, and output format

## Lab 03: Token Audit Exercise

- [ ] Token audit worksheet complete — all 4 sections filled in
- [ ] Personal mode/model cheat sheet has at least 11 rows and is saved for future reference

---

## Mindset Check

Before moving to intermediate level, confirm these behaviors:

- [ ] I read AI completions before accepting them
- [ ] I use the lowest-cost mode that solves the problem
- [ ] I can articulate what each Copilot mode is doing under the hood
- [ ] I would not commit AI-generated code without reading it line by line
