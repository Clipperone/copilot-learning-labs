# Beginner Completion Checklist

Use this checklist after completing Modules 01–03 and Labs 01–03 to confirm you are ready for the intermediate level.

---

## Module 01: Foundations

- [ ] Copilot Pro+ is active — no status bar errors
- [ ] Can switch between Ask, Edit, Plan, and Agent modes
- [ ] Can trigger and dismiss inline completions and inline chat
- [ ] Can explain — in one sentence each — when to use each mode
- [ ] Applies the four critical review questions to AI-generated code

## Lab 01: Setup and Verify

- [ ] All six success criteria from `labs/lab-01-setup-and-verify/checklist.md` are met
- [ ] Premium model access confirmed (or limitation documented)

---

## Module 02: Configuration *(available in v0.2)*

- [ ] `.vscode/settings.json` is configured for optimal Copilot behavior
- [ ] Project structure is organized to help Copilot understand context
- [ ] Linting and formatting tools are configured and running

## Lab 02: Project Configuration *(available in v0.2)*

- [ ] Configuration deliverables produced and documented

---

## Module 03: Token Optimization *(available in v0.2)*

- [ ] Can distinguish between included and premium request consumption
- [ ] Has a personal mode/model decision rule for at least 5 common task types
- [ ] Has reduced at least one unnecessary request pattern in their normal workflow

## Lab 03: Token Audit *(available in v0.2)*

- [ ] Token audit completed on a real task

---

## Mindset Check

Before moving to intermediate level, confirm these behaviors:

- [ ] I read AI completions before accepting them
- [ ] I use the lowest-cost mode that solves the problem
- [ ] I can articulate what each Copilot mode is doing under the hood
- [ ] I would not commit AI-generated code without reading it line by line
