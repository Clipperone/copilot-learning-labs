Style: Use snake_case for all identifiers in Python. Use camelCase for JavaScript and TypeScript identifiers.

Verbosity: When I ask for code, produce the code only. Do not add explanatory comments unless I ask for them.

Docstrings: When I request documentation, use Google-style docstrings with Args, Returns, and Raises sections.
