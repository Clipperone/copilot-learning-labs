## Tasks

### Task 1: Classify Your Last 5 Copilot Interactions

**Goal:** Develop cost awareness by reviewing real interactions.

**Instructions:**

1. Open `starter/audit-worksheet.md`.
2. In **Section A**, fill in the table for the last 5 Copilot interactions you can recall — or use the example interactions provided in the worksheet.
3. For each interaction, classify: mode used, model used (if known), request type (included or premium), and whether the first response was complete.

**Success criterion:** Section A table is complete with 5 rows.

---

### Task 2: Build Your Mode/Model Cheat Sheet

**Goal:** Produce a reference table you will use in practice.

**Instructions:**

1. In **Section B** of the worksheet, complete the mode/model decision table.
2. The template provides 8 starter rows. Add at least 3 more rows for task types specific to your work.
3. For each row, decide: minimum mode, preferred model, and one-sentence justification.

**Success criterion:** Section B table has at least 11 rows (8 starter + 3 personal).

---

### Task 3: Prompt Audit

**Goal:** Identify and fix prompts that required multiple follow-ups.

**Instructions:**

1. In **Section C** of the worksheet, pick 3 prompts from your recent history that required 2+ follow-ups.
2. For each, write the original prompt, identify what was missing (goal? constraints? output format?), and write the compact version.
3. If you cannot recall actual prompts, use the samples in the worksheet.

**Success criterion:** Section C has 3 rows with original, diagnosis, and compact version.

---

### Task 4: Context Hygiene Audit

**Goal:** Identify context habits to change.

**Instructions:**

1. In **Section D** of the worksheet, answer the 5 reflection questions honestly.
2. Pick the 2 habits you rate lowest and write one concrete action to change each.

**Success criterion:** Section D is complete with 2 action items identified.

## Common Failure Points

| Symptom | Likely cause | Fix |
|---------|-------------|-----|
| "I don't remember which model I used" | Model was not checked before sending | Going forward: always check the model selector before a new session |
| "All my prompts got complete first responses" | You are already writing good prompts, or you only recall the successes | Use the sample prompts in Section C to practice anyway |
| "My tasks don't fit the default rows" | That is intentional — the worksheet is a template | Add rows freely; the cheat sheet is yours |
