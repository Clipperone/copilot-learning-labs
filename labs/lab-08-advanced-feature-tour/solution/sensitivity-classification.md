## OWASP Issues Identified in `auth.py`

| # | Finding | OWASP Category | Location | Minimum fix |
|---|---------|---------------|----------|-------------|
| 1 | MD5 used for password hashing | A02 — Cryptographic Failures | `hash_password()`, `hashlib.md5(...)` line | Replace with `bcrypt.hashpw(password.encode(), bcrypt.gensalt())` from the `bcrypt` library, or use `argon2-cffi` |
| 2 | Hardcoded salt `"CHANGE_ME_SALT"` | A07 — Identification and Authentication Failures | Module level, `_SALT` assignment | Replace with a value generated by `secrets.token_hex(16)` stored securely outside the codebase — never committed |
| 3 | No input validation on `user_id` in `get_user_token()` | A03 — Injection | `get_user_token()`, `payload = f"token:{user_id}"` line | Add `if not user_id or not isinstance(user_id, str): raise ValueError("user_id must be a non-empty string")` before constructing the payload |
