# Labs

Hands-on labs for the GitHub Copilot Pro+ course. Each lab is paired with a module and includes starter files, tasks with expected outputs, success criteria, and reference solutions.

Complete labs in order unless you are following a [persona-based shortcut path](../LEARNING_PATH.md#persona-based-shortcut-paths).

## Lab Structure

Every lab folder contains:

```
README.md       ← task instructions, success criteria, failure points
checklist.md    ← completion self-assessment
starter/        ← files the learner begins with
solution/       ← reference answers
```

Labs marked *(v0.x)* are planned for a future release. See [CHANGELOG.md](../CHANGELOG.md) for the release schedule.
