# Course Overview

## GitHub Copilot Pro+ Mastery in VS Code

**Tagline:** From first autocomplete to production-grade AI-assisted workflows.

---

## What This Course Is

This is a progressive, hands-on learning program for GitHub Copilot Pro+ inside Visual Studio Code. It takes learners from initial setup through advanced multi-agent orchestration, covering every practical skill needed to use AI assistance seriously, productively, and sustainably in professional software development.

The course is a public open repository — clone it, follow the labs, adapt the templates, and apply the prompt library and agent definitions directly in your own projects.

---

## Who This Course Is For

| Persona | Starting point | Primary goal |
|---------|---------------|-------------|
| Individual developer | Little or no Copilot experience | Personal productivity and code quality |
| Engineering manager | Wants team standardization | Define team conventions and adoption plan |
| Developer productivity coach | Copilot experience, no structure | Build a repeatable training framework |
| Senior developer | Experienced user, no system | Move from ad hoc usage to deliberate AI workflows |

No prior Copilot experience is required. Familiarity with VS Code and at least one programming language is assumed.

---

## What You Will Be Able to Do

After completing this course, you will be able to:

- Configure GitHub Copilot Pro+ and VS Code for maximum productivity
- Choose the right Copilot mode — inline completion, chat, ask, edit, plan, agent — for any task
- Write effective, repeatable prompts for code generation, refactoring, debugging, testing, documentation, and security review
- Design persistent custom instructions that guide Copilot consistently across a project
- Define role-specialized agents with clear responsibilities, tool boundaries, and handoff protocols
- Orchestrate multi-agent workflows on complex tasks without polluting context or wasting requests
- Minimize premium request consumption while maximizing output quality
- Maintain architectural quality and enforce human governance of AI-generated code
- Follow a structured 7/30/60/90-day personal adoption roadmap

## Course Structure

The course is organized into **10 modules** across **4 progressive levels**, each paired with hands-on labs and reusable assets.

| Level | Modules | Estimated time | Key skill unlocked |
|-------|---------|---------------|-------------------|
| **Beginner** | 01 — Foundations | 2h | Know what Copilot does and how to evaluate its output |
| **Beginner** | 02 — Configuration | 2h | Set up VS Code and your project for optimal AI context |
| **Beginner** | 03 — Token Optimization | 2h | Make cost-aware decisions about modes and models |
| **Intermediate** | 04 — Prompt Engineering | 3h | Write reliable prompts across all coding scenarios |
| **Intermediate** | 05 — Custom Instructions | 2h | Encode project conventions into persistent guidance |
| **Advanced** | 06 — Agents | 3h | Define and operate 10 specialized agent personas |
| **Advanced** | 07 — Multi-Agent Workflows | 3h | Orchestrate agents on complex, multi-step tasks |
| **Expert** | 08 — Advanced Features | 3h | Leverage AI review, planning, terminal, and CI/CD integration |
| **Expert** | 09 — Repository Quality | 2h | Keep repositories AI-friendly and architecturally sound |
| **Expert** | 10 — Adoption Roadmap | 2h | Build and execute a 7/30/60/90-day adoption plan |
