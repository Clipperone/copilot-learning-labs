# Modules

11 progressive modules across 4 levels. Each module includes theory, exercises, and a self-assessment checklist.

| Level | Modules | Estimated time |
|-------|---------|---------------|
| Beginner | 01–03 | ~6 hours |
| Intermediate | 04–05 | ~5 hours |
| Advanced | 06–07 | ~6 hours |
| Expert | 08–11 + Capstone | ~9 hours |

Complete the modules in order. Each level builds on the previous one.
