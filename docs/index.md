## What You Will Learn

| Skill area | What you will be able to do |
|------------|----------------------------|
| **Setup and modes** | Configure Copilot Pro+ and choose the right mode for any task |
| **Prompt engineering** | Write effective, repeatable prompts for generation, refactoring, debugging, testing, and security review |
| **Custom instructions** | Design persistent instructions at global, project, and path scope |
| **Agent workflows** | Define 10 role-specialized agents with tool permissions and handoffs |
| **Multi-agent orchestration** | Orchestrate complex, multi-step tasks without wasting context |
| **Cost awareness** | Minimize premium request consumption with informed mode/model decisions |
| **Adoption planning** | Apply a 7/30/60/90-day personal and team adoption roadmap |

## Who This Course Is For

| Learner | Starting point | Primary goal |
|---------|---------------|-------------|
| Developer new to Copilot | No Copilot experience | Get productive fast with a structured foundation |
| Developer using Copilot informally | Uses completions and basic chat | Move to deliberate, repeatable workflows |
| Engineering manager | Team has Copilot, no standards | Define conventions and adoption milestones |
| Productivity coach | Copilot experience, no framework | Build a structured training program |

**Prerequisite:** Basic familiarity with VS Code and at least one programming language.
