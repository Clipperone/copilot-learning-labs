# Labs

9 hands-on labs, each paired with a module 01–09. Complete the labs in order for the best learning experience.

Each lab includes starter files and a reference solution. View them on [GitHub](https://github.com/Clipperone/copilot-learning-labs/tree/main/labs).

> **Modules 10 and 11** do not have separate labs. Module 10 applies its skills directly through the [Capstone](../capstone). Module 11 (Platform & GitHub.com Integration) runs in-module exercises on github.com, the `gh copilot` CLI, and the Copilot coding agent — places where a local lab folder would be artificial. See the [Module 11 page]({{ '/modules/11-platform-integration' | relative_url }}) for details.
